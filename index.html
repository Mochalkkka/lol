Команды Git
Команда Описание
git init - is a command used in Git to initialize a new Git repository
git clone url - is a command used in the Git version control system to make a local copy of a remote repository located at the specified URL
git add file - is a commonly used command in version control systems such as Git
git add . - is a Git command that adds all the changes in the current directory and its subdirectories to the staging area
git commit -m "comment" - is used in the Git version control system to create a new commit with a commit message
git commit -a -m "comment" - is a terminal command used in Git to commit changes to the local repository with a message describing the changes made
git push url - is used in Git version control system to push changes made in a local repository to a remote repository located at the specified URL.
git status - is a command used in the Git version control system
